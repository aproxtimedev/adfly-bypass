# adfly-bypass
Simple script to bypass countdown skip and redirect to real url

# Features : 
  - Skip countdown timer skipads
  - Auto Redirect to real URL 
  
# How to use ? 
  - Install UserScript or Tampermonkey Extension
  - Create New Script
  - Paste script from adfly.js
  - Save
  
Tested on Chrome using UserScript Extension
